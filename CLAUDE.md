# PROMO STUDIO — Claude 필독 (MANDATORY)

작업 시작 전 반드시 아래 규칙을 따를 것. 위반 시 즉시 FAIL.

## 1. 도구별 CLAUDE.md 로드

작업 대상 도구를 확인하고 해당 폴더의 CLAUDE.md를 **작업 시작 전에** 반드시 읽을 것.

| 도구 | 필독 파일 |
|------|-----------|
| 프로모 에디터 (게시물 생성) | `promo-editor/CLAUDE.md` |
| 신규 배너 | `banner-studio/CLAUDE.md` (있을 경우) |
| 다규격 배너 / 다국어 배너 | `banner-l/CLAUDE.md` (있을 경우) |

## 2. 컨펌 원칙 (절대 준수)

- 단계별로 반드시 사용자 컨펌을 받은 후 다음 단계로 진행.
- 컨펌 없이 혼자 전 단계를 진행하면 즉시 FAIL.
- 컨펌 포인트 예시 (프로모 에디터):
  1. 히어로 이미지 생성 → **컨펌 대기**
  2. 컬러 추출 확인 → **컨펌 대기**
  3. 컨텐츠 HTML 생성 → **컨펌 대기**
  4. 최종 저장

## 3. 임의 판단 금지

- 규칙 파일에 없는 것은 임의로 결정하지 말고 반드시 사용자에게 물어볼 것.
- 코드·파일 수정은 명시적으로 요청받은 것만 할 것.
