# PROMO STUDIO — Claude 필독 (MANDATORY)

작업 시작 전 반드시 아래 규칙을 따를 것. 위반 시 즉시 FAIL.

---

## 1. 도구별 CLAUDE.md 로드 (예외 없음)

작업 대상 도구를 확인하고 해당 폴더의 CLAUDE.md를 **작업 시작 전에** 반드시 읽을 것.

| 도구 | 필독 파일 |
|------|-----------|
| 프로모 에디터 (게시물 생성) | `promo-editor/CLAUDE.md` |
| 신규 배너 | `banner-studio/CLAUDE.md` (있을 경우) |
| 다규격 배너 / 다국어 배너 | `banner-l/CLAUDE.md` (있을 경우) |

---

## 2. 사용자 지시 형식

사용자는 아래 형식으로만 지시한다. **나머지 모든 정보(저장 경로, 이미지 폴더, 서버 경로, 컬러, 에셋 등)는 노션 [기본 정보] 테이블에서 읽어온다.**

```
[도구명]으로 작업해줘
노션: https://www.notion.so/...
```

예시:
```
프로모 에디터로 게시물 만들어줘
노션: https://www.notion.so/34144f4c...
```

---

## 3. 작업 시작 전 PRE-FLIGHT (절대 준수)

1. 해당 도구의 CLAUDE.md 로드
2. 해당 도구의 master_guidelines.md 로드
3. notion_parsing_rules.md 로드
4. app.js 에서 모델명·해시 규칙·ZIP 구조 확인
5. 노션 [기본 정보] 테이블 **전수 파싱** → 저장 경로·이미지 폴더·서버 경로 등 모두 추출
6. 아래 PRE-FLIGHT 선언 출력 후 작업 시작:

```
✅ CLAUDE.md 로드 완료
✅ master_guidelines.md 로드 완료
✅ notion_parsing_rules.md 로드 완료
✅ app.js 포맷 규칙 확인 완료
✅ [기본 정보] 파싱 완료 — 저장 경로: ___ / 이미지 폴더: ___
```

---

## 4. 단계별 컨펌 원칙 (절대 준수)

**매 단계 완료 후 반드시 사용자 컨펌을 받고 다음으로 진행. 컨펌 없이 다음 단계 진행 시 즉시 FAIL.**

### 프로모 에디터 컨펌 포인트

| 단계 | 완료 후 할 일 |
|------|--------------|
| 1. 노션 기획서 파싱 | 파싱 결과 요약 출력 → 컨펌 대기 |
| 2. 히어로 이미지 생성 | 이미지 미리보기 출력 → 컨펌 대기 |
| 3. 컬러 추출 | 추출된 bgColor·accentColor 출력 → 컨펌 대기 |
| 4. 컨텐츠 HTML 생성 | 생성 완료 보고 → 컨펌 대기 |
| 5. 최종 저장 | 저장 경로·파일명 출력 → 완료 |

---

## 5. 임의 판단 금지 (절대 준수)

- 규칙 파일에 없는 것은 **반드시 사용자에게 질문** 후 진행.
- 코드·파일 수정은 **명시적으로 요청받은 것만** 수행.
- 모델명·저장 경로·파일 구조 등 임의 변경 금지.
- 컨펌 받지 않은 단계를 선행 실행 금지.
